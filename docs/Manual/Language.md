# Language
This page serves as a dictionary for any Industry-specific jargon.

- **FPS:** Abreviation for "First Person Shooter". A game genre that includes navigating an environment from a first-person perspective. The name originally described a genre of violent gun games, but now the same core movement and mechanics describe a wide variety of nonviolent games as well. Nonviolent games tend to describe themselves as simply "First Person" games, but they still get sorted under the same "FPS" category by their mechanics.
- **Mechanic:** A discrete function or feature of a game. Often used to point to a function or feature, or groups of them as a reference in a larger discussion. Usefull in discussing the form and flow of interactive concepts.
